# Physics Lab

一个基于 HTML5 Canvas 的 2D 物理沙盒模拟器，用于物理教学和实验演示。

> 纯浏览器端运行，无需安装任何依赖，打开即用。

## 在线体验

直接打开 `physics_lab.html` 即可使用。

## 功能

- **2D 物理引擎**：重力、摩擦力、弹力、空气阻力
- **多种物体**：小球（含质点模式）、板砖
- **碰撞系统**：球-球、球-砖、砖-砖弹性/非弹性碰撞
- **电磁场模拟**：电场力、磁场力（洛伦兹力）
- **传送带**：可设置速度和方向的传送带
- **轨道绘制**：自由绘制碰撞轨道
- **圆形轨道**：环形碰撞边界
- **力分析**：实时显示重力、支持力、摩擦力、合力等
- **暂停编辑**：暂停时可修改物体的速度和加速度
- **对象属性**：选中物体后可在右侧面板调整大小、质量等参数

## 操作

| 操作 | 功能 |
|------|------|
| 左键单击 | 选择对象 |
| 左键拖拽 | 移动选中对象 |
| 滚轮 | 缩放画布 |
| Shift+左键 | 平移画布 |
| 空白处拖拽 | 框选多条线段 |
| Delete | 删除选中对象 |

### 快捷键

| 键 | 功能 |
|----|------|
| `1` | 选择工具 |
| `2` | 绘制轨道 |
| `3` | 平移工具 |
| `Space` | 暂停/继续 |
| `G` | 生成小球 |
| `B` | 生成板砖 |

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript (ES6+)
- Canvas 2D 渲染
- 固定时间步长物理引擎 (240Hz)
- 半隐式欧拉积分
- SAT 分离轴定理碰撞检测
